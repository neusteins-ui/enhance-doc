Guide

Color
Best for letterheads, stamps, photos, or any color content. Images return as JPG.
Grayscale
Standard black-ink docs. Smaller file size, clean output. Images return as PNG.
Black & White
Maximum contrast for text-only scans. Smallest file, sharpest text. Images return as PNG.
Higher DPI
Use 400–600 for very small print, signatures, or archival quality. Applies to PDF output.
Supported Inputs
PDF, JPG, PNG, TIFF, BMP, WEBP. Multi-page PDFs are fully supported.
Output Formats
PDFs stay as PDF. Images are returned as JPG (color) or PNG (grayscale/BW).
